Donchian策略
==========

设置一条阻力线和一条支撑线，阻力线由过去N天的最高价的最大值形成；支撑线由过去N天的最低价的最小值形成。

13.1计算方法
--------

.. math:: 上轨\  = \ Max\left. （最高价，n \right.），\ n日最高价的最大值

.. math:: 下轨\  = \ Min\left. （最低价，n \right.），n日最低价的最小值

.. math:: 中轨\  = \ \frac{(上轨 + 下轨)}{2}

13.2策略逻辑
--------

（1）空仓：价格突破唐奇安上轨开多，价格跌破唐奇安下轨开空；

（2）多头：价格跌破唐奇安中轨平多；

（3）空头：价格突破唐奇安中轨平空。
