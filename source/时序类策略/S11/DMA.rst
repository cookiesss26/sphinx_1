DMA策略
=====

DMA指标即所谓平均线差指标，是股市分析技术中的一种中短期指标。常被用于大盘指数和个股分析的研究判断，DMA指标也是一种运用较多的趋势型分析指标。

12.1 计算方法
---------

.. math:: DMA = MA1 - MA2

.. math:: AMA = \frac{\sum_{i = 1}^{i = M}{DMA}}{M}

其中，MA1为短期（S日）的移动均线，MA2为长期（L日）移动均线，AMA为DMA的M日移动平均线。

12.2策略逻辑
--------

做多：DMA向上交叉其平均线AMA；

做空：DMA向下交叉其平均线AMA 。
